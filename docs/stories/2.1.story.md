## Story

Como equipe ON Office, quero a fundação do design system dark premium (tokens, utilities,
fonte de título), para que todas as telas possam ser migradas de forma consistente.

## Acceptance Criteria

1. `src/index.css` — `:root` dark-first: background `#0A0A0C`, card `#121214`, foreground
   `#EDEDEF`, muted-fg ≥4.5:1, border hairline, ring lime, radius 1.25rem; tokens sidebar
   coerentes. (O bloco `.dark` é unificado — sistema é dark-only por decisão do usuário.)
2. Utilities novas: `on-glow`/`on-glow-sm` (lime glow), `on-glass` (blur 16px + fallback
   `@supports`), `on-card` dark (borda branca 8%, elevação por borda+sombra preta),
   `on-button` com glow no hover, `on-tile`/`on-pill` mantidos, `on-bento` (grid base).
3. `prefers-reduced-motion: reduce` desativa transições/animações globais.
4. Outfit carregada no `index.html` + `font-outfit` no tailwind config; headings da base
   (`h1-h3` via utility `font-outfit`) sem quebrar Work Sans no corpo.
5. Labels de baixo contraste da fundação corrigidos (gray-400 → token ≥4.5:1).
6. Gates: tsc + eslint (arquivos da story) + build verdes.
