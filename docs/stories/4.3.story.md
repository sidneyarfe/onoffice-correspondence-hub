## Acceptance Criteria

1. `CrmDealSheet` (shadcn `Sheet`) aberto ao clicar num card/linha, sincronizado com os dados
   frescos do negócio.
2. **Contato**: nome, e-mail (mailto), telefone, origem, campanha (UTM).
3. **WhatsApp**: botão abre `wa.me` com o telefone normalizado (DDI 55 quando faltar) e mensagem pré-pronta.
4. **Etapa**: `Select` move o negócio (reusa `moverEtapa`).
5. **Tags**: chips removíveis + dropdown para aplicar tags existentes (`vincularTagNegocio`) e
   **criar nova tag** inline (`criarTag` → vincula). Tags valem para o negócio.
6. **Atividades**: timeline (ícone por tipo, data pt-BR), criação (tipo + descrição) via
   `criarAtividade`, e marcar concluída para tipos acionáveis (`alternarAtividade`).
7. Botão "Contratação" abre o fluxo da Story 4.4.
8. Gates: `tsc --noEmit` ✅, `eslint` ✅.
